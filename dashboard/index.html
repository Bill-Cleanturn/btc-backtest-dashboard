  • 1위 전략 자본곡선 차트(초기 10,000)
  • 상위 12개 최종자본 비교 차트
  • 각 전략의 상세 거래내역/차트는 아래 테이블의 보기 버튼으로 확인
  • 실행 리포트(요약)

설명

  • 평균수익(%/trade): 한 번 거래할 때 평균적으로 몇 % 수익/손실이 나는지
  • 승률: 수익으로 끝난 거래 비율
  • 거래 수: 샘플 개수(너무 작으면 신뢰도 낮음)
  • 반전진입: 기존 전략의 반대 방향으로 진입
  • 최종금액: 초기 10,000으로 시작했을 때 거래를 순서대로 반영한 최종 자산
